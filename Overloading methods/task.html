Given a method that takes a long value and returns the max value of the long type minus the given number.

public static long getMaxMinusCurrent(long val) {
    return Long.MAX_VALUE - val;
}

Overload this method with two new methods. The first one should take an int and return the max value of the int type minus the given value. The second one should take a short value and return the max value of the short type minus the given value.

For both methods, the type of result value must be the same as the type of the parameter.


Sample Input:
long
10
Sample Output:
9223372036854775797

Sample Input:
int
8
Sample Output:
2147483639


Memory limit: 256 MB
Time limit: 8 seconds
