Getting input Open on JetBrains Academy

The Scanner class is widely used to parse text to strings and it is by far the simplest way to get input in Java.

In this task, you simply need to use method nextLine to read the whole line of input. The trick is to use the Scannerclass without the import statement.

The application will receive a string from the method, and output it in the corresponding format.


Sample Input:
Test
Sample Output:
The input string: Test
